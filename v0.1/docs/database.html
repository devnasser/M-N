📊 الجداول الرئيسية

👥 جدول المستخدمين (users)

  • id - المعرف الفريد
  • name - الاسم
  • email - البريد الإلكتروني
  • phone - رقم الهاتف
  • password - كلمة المرور
  • role - الدور (admin, buyer, shop, driver, technician)
  • status - الحالة (active, inactive, suspended)
  • preferred_language - اللغة المفضلة
  • avatar - الصورة الشخصية
  • created_at, updated_at - التواريخ

📂 جدول الفئات (categories)

  • id - المعرف الفريد
  • name - الاسم بالعربية
  • name_en - الاسم بالإنجليزية
  • slug - الرابط المختصر
  • description - الوصف
  • description_en - الوصف بالإنجليزية
  • image - صورة الفئة
  • icon - الأيقونة
  • color - اللون
  • parent_id - الفئة الأب
  • sort_order - ترتيب العرض
  • is_active - نشط/غير نشط

🏪 جدول المتاجر (shops)

  • id - المعرف الفريد
  • user_id - معرف المستخدم
  • name - اسم المتجر
  • name_en - اسم المتجر بالإنجليزية
  • description - وصف المتجر
  • logo - شعار المتجر
  • cover_image - صورة الغلاف
  • phone - رقم الهاتف
  • email - البريد الإلكتروني
  • website - الموقع الإلكتروني
  • address - العنوان
  • latitude, longitude - الإحداثيات
  • city - المدينة
  • region - المنطقة
  • status - الحالة
  • is_verified - موثق
  • is_featured - مميز
  • rating - التقييم

🛍️ جدول المنتجات (products)

  • id - المعرف الفريد
  • shop_id - معرف المتجر
  • category_id - معرف الفئة
  • name - اسم المنتج
  • name_en - اسم المنتج بالإنجليزية
  • slug - الرابط المختصر
  • description - الوصف
  • sku - رمز المنتج
  • brand - العلامة التجارية
  • model - الموديل
  • price - السعر
  • sale_price - سعر البيع
  • stock_quantity - الكمية المتوفرة
  • images - الصور (JSON)
  • specifications - المواصفات (JSON)
  • compatibility - التوافق (JSON)
  • is_active - نشط/غير نشط
  • is_featured - مميز
  • rating_average - متوسط التقييم
  • view_count - عدد المشاهدات

📦 جدول الطلبات (orders)

  • id - المعرف الفريد
  • order_number - رقم الطلب
  • user_id - معرف المستخدم
  • shop_id - معرف المتجر
  • driver_id - معرف السائق
  • status - الحالة (pending, confirmed, shipped, delivered, cancelled)
  • payment_status - حالة الدفع
  • payment_method - طريقة الدفع
  • subtotal - المجموع الفرعي
  • tax_amount - الضريبة
  • shipping_amount - رسوم الشحن
  • total_amount - المجموع الكلي
  • shipping_address - عنوان الشحن
  • billing_address - عنوان الفواتير
  • notes - الملاحظات
  • estimated_delivery - موعد التوصيل المتوقع

📋 جدول عناصر الطلبات (order_items)

  • id - المعرف الفريد
  • order_id - معرف الطلب
  • product_id - معرف المنتج
  • quantity - الكمية
  • unit_price - سعر الوحدة
  • total_price - السعر الإجمالي
  • product_name - اسم المنتج
  • product_sku - رمز المنتج
